Query used to create the list of ontologies in LOV: https://lov.linkeddata.es/dataset/lov/sparql
```
PREFIX vann:<http://purl.org/vocab/vann/>
PREFIX voaf:<http://purl.org/vocommons/voaf#>

SELECT DISTINCT  ?voc ?download_url ?date {
 	GRAPH <https://lov.linkeddata.es/dataset/lov>{
 	 	?voc <http://www.w3.org/ns/dcat#distribution> ?download_url.
    ?download_url <http://purl.org/dc/terms/issued> ?date
}}
```
Blank nodes on the distribution were removed.
